# PDF QnA Bot

A simple question-answering bot that allows users to upload a PDF and ask questions based on its content using LangChain and LLMs.

## Features
- Upload PDF files
- Ask questions based on uploaded document
- Chunking and semantic search
- LLM-based answers

## Tech Stack
- Python
- LangChain
- PyPDF
- FAISS
- Streamlit

## Setup
```bash
pip install -r requirements.txt
streamlit run app.py
